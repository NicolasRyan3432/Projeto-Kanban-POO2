# Projeto-Kanban-POO2
App Kanban em Java para a conclusão da disciplina de POO2

O objetivo deste aplicativo é de ajudar o usuário a organizar as suas atividades do seu dia-a-dia. 
Foi construído utilizando **Java Swing** para a interface gráfica, **Maven** para gerenciamento de dependências e **MySQL** para o banco de dados.

## Como O Kanban funciona 
O sistema é dividido em três colunas:
**A FAZER**: é destinado as atividades que SERÃO FEITAS, ou seja, tudo que deseja ser feito deverá estar nessa coluna e as notas são criadas lá.
**SENDO FEITO**: é destinado as atividades que ESTÃO SENDO REALIZADAS, ou seja, ao começar a atividade, ela passa para essa coluna e fica até ser concluída. Podendo ser modificada a nota de acordo com a necessidade.
**CONCLUÍDO**: é destinado as atividades que já FOREM CONCLUÍDAS.
